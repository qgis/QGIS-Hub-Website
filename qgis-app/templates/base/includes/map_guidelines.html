Important Guidelines for Uploading Your Map

Before proceeding with the upload, please ensure that your map adheres to the following guidelines:

  • The map is created using QGIS.
  • The map includes the "Made with QGIS" logo (included by default in QGIS SVG resources).
  • The map is in a supported image format (e.g. PNG, JPEG).
  • The map is not a screenshot.
  • The map file size does not exceed 10MB.
  • The map name provided in the form is descriptive and relevant.
  • The description provided in the form includes a concise narrative explaining how QGIS was utilized to create the map.
