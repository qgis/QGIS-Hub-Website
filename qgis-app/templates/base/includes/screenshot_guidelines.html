Important Guidelines for Uploading Your QGIS Screenshot

Before proceeding with the upload, please ensure that your screenshot adheres to the following guidelines:

  • The screenshot clearly shows the QGIS interface and relevant content.
  • The screenshot is in a supported image format (e.g. PNG, JPEG, GIF, SVG).
  • The screenshot does not contain sensitive or personal information.
  • The file size does not exceed 10MB.
  • The file name provided in the form is descriptive and relevant.
  • The description provided in the form includes a concise narrative explaining what is shown in the screenshot and how QGIS was used.
